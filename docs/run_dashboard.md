# Run the Interactive Dashboard (Dash)

This project includes a local interactive dashboard built with **Dash + Plotly**.

## 1) Install dependencies

From the project root:

```bash
pip install -r requirements.txt
```

## 2) Run the app

```bash
python dash_app/app.py
```

## 3) Open in your browser

Go to:

- http://127.0.0.1:8050

## Notes
- The app reads: `data/processed/superstore_cleaned.csv`
- Use the top filters (Year, Region, Segment, Category) to slice all charts.
